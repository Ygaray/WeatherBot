<objective>
Make every config snapshot truly immutable by adding `frozen=True` to the existing
`ConfigDict(extra="forbid")` on ALL FIVE config models — `Config`, `Location`, `Schedule`,
`Reliability`, `WebhookIdentity` (D-02). This is the type-enforced half of the "immutable
snapshot" guarantee: a job that tries to mutate the shared live config raises immediately
instead of silently corrupting what every other thread reads.

Purpose: A `ConfigHolder` hands out a shared reference, not a copy (Plan 03). That is only
safe if the reference cannot be mutated in place — `frozen=True` is what makes shared,
lock-free reads correct. Without it, "immutable snapshot" would be convention, not guarantee.
Output: Five one-field edits to `weatherbot/config/models.py`, turning the Plan 01
`test_frozen_rejects_mutation` cases GREEN while keeping all 215 existing tests green.
</objective>
