| config.toml -> BotConfig | operator-edited config crosses into the running process at load |
| daemon thread -> bot thread | the new panel_channel_id flows into the isolated bot thread; a bad value must not crash the scheduler |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
