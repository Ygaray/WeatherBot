<objective>
Land the PRIMARY offline fatal gate (D-07, HARD-STARTUP-01): make `run()` call the full `validate_config_and_templates(args.config)` before `run_daemon`, catching the SAME four-exception set `check-config` catches, and route any failure into a single `_fatal_config_exit(settings, reason, detail)` helper (D-08) that best-effort-alerts once, stamps the durable health row with CONFIG_INVALID (D-02), and returns non-zero. Purpose: A misconfigured daemon can no longer boot green and silently drop every briefing; it fails loudly at boot exactly like check-config, and the human hears about it on Discord + the OS layer.
Output: `_fatal_config_exit` helper + the `run` boot-validate gate in cli.py.
</objective>

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| operator config/template → `run` boot validator | Local operator TOML/Jinja; the validator is the fail-loud boundary (ASVS V5). A permanent fault must exit non-zero, not boot green. |
| fatal alert `detail` → Discord/log | Outcome-only; must never carry a filesystem path or secret. |
| best-effort `channel.send` → shutdown | A hung/failed send must not delay or mask the non-zero exit. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-29-10 | Information Disclosure | `_fatal_config_exit` alert/log detail | high | mitigate | `detail=type(exc).__name__` (never `str(exc)`); Task 1 acceptance greps out `str(exc)` in the helper; a config error can embed a path (T-04-01, ASVS V7). |
| T-29-11 | Denial of Service | best-effort fatal `channel.send` | medium | mitigate | try/except around the send; the non-zero return is computed independent of send success so a hung alert never blocks the exit. |
| T-29-12 | Elevation/Tampering (lifecycle) | green boot on bad config | high | mitigate | The full-validator gate + parity test guarantee `run` rejects exactly what `check-config` rejects, closing F05 (no silent green boot). |
