## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| filesystem (`.env`) → process | Bearer-style secrets (API key, webhook URL) enter the process here; must not leak back to git, config, or logs |
| `config.toml` (hand-edited) → process | Untrusted-shape user input validated by Pydantic at load |
| PyPI → developer machine | Third-party packages installed via uv (supply chain) |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-01-02 | Tampering | Malformed `config.toml` crashes or misconfigures | mitigate | Pydantic validates `Config` at load; missing/typed-wrong fields raise `ValidationError` (fail-loud) — tested |
| T-01-SC | Tampering | uv/PyPI package installs | mitigate | Package Legitimacy Audit ([ASSUMED], slopcheck unavailable) + blocking human-verify checkpoint (Task 4) before relying on the install |
</threat_model>
