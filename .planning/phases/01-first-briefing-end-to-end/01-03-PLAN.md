<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| OpenWeather payload → SQLite `raw_json` | Persisted response must not carry the appid/secret or request URL |
| user-edited template file → renderer | Untrusted template text must not enable format-string injection |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Information Disclosure | Secret leak via persisted payload | mitigate | `persist` stores ONLY response JSON, never the request URL (which holds `appid`); test asserts no `appid` in any stored `raw_json` |
| T-03-02 | Tampering / DoS | Format-string / template injection on a user-editable template | mitigate | Guarded `_Safe`-mapping substitution over a flat str→str whitelist; never `eval`/`str.format(**obj)` — tested |
| T-03-03 | Denial of Service | Missing placeholder crashes the render at send time | mitigate | Missing key renders visibly (`{key}`) without raising; Phase 2 adds strict validation |
</threat_model>
